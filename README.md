# Team USHIET - DATATHON 2026

## Quick Start

**Pull LFS**
```bash
git lfs pull
```

**Unzip data archive**
```bash
mkdir assets/data
unzip assets/datathon-2026-round-1.zip -d assets/data
```

**Install Python Envrionment**
```bash
python -m venv .venv & . .venv/Scripts/activate
pip install -r requirements.txt
```
